# 永不消逝 · 混剪视频脚本
## 糯糯 × 菲比 / V × 强尼银手 替换混剪

**视频时长目标：4分30秒 ～ 5分钟**
**风格参考：** CP2077官方预告片 + 音乐MV叙事节奏
**配乐建议：** "Never Fade Away"（原曲）/ 中文重混版 / 或电子赛博朋克曲风

### 【一】0:20 — 0:55 | 弥虹市 / 糯糯登场

```

→ 镜头下沉，穿过广告无人机，落进底层区
→ 街道特写：污水、电路纹路灯、人群

[切]
糯糯侧脸特写——银灰夹克，蓝灰长发，耳后神经接口微微发光
她走在街道上，脚步不快，不乱，目视前方
脸上没有表情，但眼神很清醒

[旁白·糯糯声音，很平静]
"弥虹市有几百万人想出人头地。
```

## 制作备注

### 画面节奏说明

| 段落 | 节奏 | 情绪 |
|---|---|---|
| 序～第二幕 | 缓，建立世界 | 冷静，孤独 |
| 第三幕 | 骤快，剪切碎 | 紧张，心碎 |
| 第四幕 | 反差，突然荒诞 | 意外，有点好笑 |
| 第五幕 | 中等，压迫感 | 严肃，倒计时感 |
| 第六幕 | 放慢，碎片闪回 | 哀伤，悬念 |
| 第七幕 | 拉起，战斗快切 | 肾上腺素 |
| 第八～九幕 | 再次放慢 | 心碎，释然 |

### 台词配音方向

- **糯糯配音**：语调很平，不起伏，但每句话有分量。不用气声，用真声，很轻，偶尔停顿。
- **菲比配音**：前半段活泼、跳脱，有能量；后半段在关键时刻要掉下来，第一次听到自己"真名"时声音要微微颤。

### 对应CP2077关键场景

| 混剪段落 | 对应CP2077场景 |
|---|---|
| 小鹿牺牲 | Jackie Welles的死亡，临死前说"为我活下去" |
| 菲比第一次出现 | Johnny第一次在V脑子里出现，狂妄宣告 |
| 老维诊断 | Viktor Vector诊断V，"你只有几周" |
| 菲比的记忆碎片 | V被迫体验Johnny的记忆（Alt和Arasaka） |
| 星耀高塔决战 | 攻入阿拉萨卡塔楼，对战Adam Smasher |
| 数字深海选择 | 最终选择：Johnny取走V的身体，或V留下 |
| "永不消逝" | "Never Fade Away" —— 全片标志性台词 |
